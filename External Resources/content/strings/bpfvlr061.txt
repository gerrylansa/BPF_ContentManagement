IP Address Monitoring

The portal can monitor IP addresses for suspicious request behaviour. You can are able to enabled monitoring fo IP addresses and set a limit on the number of requests from an IP that appear to be invalid. A background server process can also be controlled and configured to check if IP addresses that have been banned and clear is ban after a defined amount of time.
